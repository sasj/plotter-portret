# plotter 

- A3 formaat = 297 x 420 mm tekengebied
- Verschillende pennen en papier
- nog wat extra software om tekening naar de machine te sturen. 


# opdracht: teken een portret van een ander

- Maak duo's 
- materialen: A4 + pen / marker / stift

1 persoon begint. Je gaat door middel van alleen rechte lijnen de andere portretteren.  
Kijk naar licht en donker, teken lijnen voor donkere gebieden.
